Automatically make any iframe embeds responsive.

== Description ==

This plugin will magically make any iframe embeds responsive.

= Live Example =

[Visit the test page](https://codepen.io/paulmaloney/full/LVVJMw/) A few examples of iframes being made magically responsive

= Multisite Compatibility =

Multisite compatiable

== Installation ==

1. Upload the plugin to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You're good to go
